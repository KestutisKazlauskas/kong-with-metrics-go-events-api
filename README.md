# Go api example

command for seting environment variable from .env file
```shell
source export.sh
```

Running the go api
```shell
go run main.go
```

Building the go api
```shell
docker build -t kstutisk4/go-api-example:0.1.x .
```

Pushhing to repository
```shell
docker push  kstutisk4/go-api-example:0.1.x
```
